## Project Overview

This project is a portfolio website inspired by Netflix's UI/UX, built using modern web technologies. It demonstrates:

- **Next.js** for server-side rendering and routing  
- **TypeScript** for type safety  
- **Reusable components** for scalable UI development  
- **Cloudinary** for optimized image delivery  
- **Zustand** for simple and effective state management  
- Clean and maintainable code structure  

## Features

- Netflix-style horizontal scroll sections  
- Hero banners with optimized media backgrounds  
- Responsive design for all device sizes  
- State management using Zustand for simplicity and performance  
- Modular and reusable React components  

## Tech Stack

- Next.js  
- React  
- TypeScript  
- Zustand  
- Cloudinary  
- CSS Modules  

## Getting Started

1. Clone the repository  
2. Install dependencies with `npm install` or `yarn`  
3. Run the development server with `npm run dev` or `yarn dev`  
4. Customize data and components to fit your portfolio needs  

## Why This Approach?

- Leverages Next.js for SEO and performance benefits  
- Uses TypeScript for better developer experience and fewer runtime errors  
- Applies state management only where necessary with Zustand  
- Optimizes images with Cloudinary for faster loading times  
- Prioritizes clean, scalable architecture for easier maintenance
